# atprotocol-apps

A curated, categorized, and alphabetically sorted directory of applications, services, and tools built on the **Authenticated Transfer (AT) Protocol**.

## Project Overview
This project serves as a centralized hub for discovering the growing "Atmosphere" (the AT Protocol ecosystem). It organizes various clients and platforms—ranging from microblogging and long-form writing to developer infrastructure—into a clean, accessible HTML interface.

## Key Features
- **Logical Categorization:** Apps are grouped into 8 distinct domains (e.g., *Lifestyle & Events*, *Social Bookmarking & Curation*, *Developer Tools*).
- **Alphabetical Organization:** Both categories and the applications within them are strictly ordered alphabetically for rapid scanning.
- **Decentralized Focus:** Highlights tools that leverage account portability and data ownership.
- **User-Centric Design:**
    - Responsive HTML5 structure.
    - All external links utilize `target="_blank"` for non-disruptive browsing.
    - Native tooltips via `title` attributes for improved accessibility and context.

## Technical Specifications
- **Languages:** HTML5, CSS (Internal/Inline).
- **Architecture:** Minimalist, single-page application (SPA) style utilizing native `<details>` and `<summary>` elements for accordion-style navigation without the need for heavy JavaScript.

## File Structure
- `index.html`: The primary entry point containing the directory structure and metadata.
- `CONTRIBUTING.md`: Guidelines for adding new apps.
